   - Supervised Learning (labeled data) vs Unsupervised (pattern discovery)

   - **Regression:** Numerical output (e.g., test score 71, 78)
   - **Classification:** Categorical output (e.g., Succeed/Fail)
   - **Three Algorithm Pillars:**
     1. Linear Models: Best fit line equations
     2. Tree-Based: Decision Trees, Random Forest (salad analogy)

   - Data partitioning: Training vs Testing sets
   - **Confusion Matrix:**
     |  | Predicted: Positive | Predicted: Negative |
     |---|---------------------|---------------------|
     | **Actual: Positive** | True Positive (TP) | False Negative (FN) - Type II |
     | **Actual: Negative** | False Positive (FP) - Type I | True Negative (TN) |
   - Error trade-offs: Marketing (Type I acceptable) vs Fraud (Type II catastrophic)
   - **Addressing Overfitting:**
     - L1/L2 Regularization for linear models
     - Tree Depth limits for decision trees

7. Unsupervised Learning: Advanced Clustering Techniques
   - **DBSCAN:** Density-based, finds non-spherical clusters, handles outliers
   - **K-Means:** Fast, assumes spherical clusters of similar size
   - **Hierarchical:** Agglomerative (bottom-up), creates Dendrogram visualization
